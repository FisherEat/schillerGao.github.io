Storyboard实现导航

storyboard实现简单的导航，无须一行代码。步骤：

  1. 新建一个工程，在Main.storyboard中将原来的Viewcontroller删除。

  2. 拖入一个NavigationController，删除不必要的那个TableViewController。将该navigationcontroller设置为initial controller ，push到下个viewcontroller时设置为rootviewcontroller

  3. 拖入一个新的ViewController，将该viewcontroller的class设置为自定义的VC。
